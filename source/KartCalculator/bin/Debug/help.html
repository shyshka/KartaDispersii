Выбор типа карт для контроля рассеяния

 

Постановка задачи

Рассматривается процесс с множеством р контролируемых параметров Х1, Х2… Хр. Через определенные равные промежутки времени делается выборка из n наблюдений: xijt – результат i-го наблюдения (i = 1…n) по j-му параметру (j = 1…p) для выборки с номером t (t = 1…m, m – количество выборок, взятых для анализа отлаженного процесса). По этим данным могут быть найдены вектор средних значений m и оценка ковариационной матрицы S процесса.

Предполагается, что в дальнейшем происходит нарушение процесса: его рассеяние  увеличивается и достигает значения S1, причем определители ковариационных матриц в нарушенном  и  исходном  процессе  (обобщенные  дисперсии)  связаны соотношением |S1| = d|S|,  (d=1.25 - 2).

Задача: определить это нарушение по результатам мониторинга как можно быстрее. С помощью каких инструментов и при каких значениях параметров этих инструментов это можно сделать? Предполагается использование трех инструментов: карты обобщенной дисперсии, кумулятивных сумм для обобщенных дисперсий и экспоненциально взвешенных скользящих средних для обобщенных дисперсий.

Параметры карт обобщенной дисперсии и кумулятивных сумм рассчитываются аналитически, для экспоненциально взвешенных скользящих средних необходимы испытания.

Для сравнения чувствительности карт к увеличению дисперсии используется средняя длина серий (сдс) – количество выборок от момента нарушения до момента обнаружения этого нарушения. Здесь также проводятся испытания.

В качестве файла данных загружаются параметры реального исследуемого процесса.

Реальное обнаружение нарушений проводиться с помощью программы  мониторинга с построением выбранного типа карт, которая будет только считывать полученные в этой программе результаты и использовать их для мониторинга.

4.Моделирование увеличения рассеяния

Предполагается, что в дальнейшем происходит нарушение процесса: его рассеяние  увеличивается и достигает значения S1, причем определители ковариационных матриц в нарушенном  и  исходном  процессе  (обобщенные  дисперсии)  связаны соотношением |S1| = d|S|,  (d=1.25 - 2).

Таким образом, надо смоделировать выборки, как и в п.2, но с ковариационной матрицей S1, каждый элемент которой определяется как элемент матрицы из п.2, умноженный  на d1/p  (т.е корень р-й степени из d).
